Tutorial

This section is the dashboard of the transmitted light datatable. This is composed of two main sections:
  • The dropdown buttons: this is where you will input the observations of properties of the thin section you are observing under transmitted light,
  • The Advanced settings button and Reset button. The first contains settings for the datatable. The second allows you to reset your previous selections for the dropdowns to start anew.

This is the heart of the transmitted light page: the datatable. Each row corresponds to a single mineral, with each column intercepting that row corresponding to an observable property of that mineral. There are a few features in-built into the datatable:
  • The Show more button, on the top-right of the datatable, reveals a list of hidden columns; these are properties which are not so frequently useful and are therefore hidden by default,
  • Hovering (or clicking on mobile) over each mineral name (in the first column) with make appear one or two pictures of that mineral, observed under transmitted light. (See below)

Below the main datatable, there is a secondary datatable. Here will appear minerals which do not satisfy all of the specified properties exactly: the suggested minerals is based on how many properties are matching, along with a tolerance in case two properties are easily confused when observing the mineral under the microscope.

The suggested minerals are displayed within a tab containing, from left to right:
  • A colour tag. There are three possible colours (see figure above): green, orange and red. They denote a mineral which respectively matches the searching criteria well, approximatively and badly,
  • A mineral name. This is the name of the suggested mineral,
  • An Add to datatable button (). Clicking this will add this mineral into the datatable of matching minerals,
  • A delete suggested mineral button (). Clicking this will remove the suggested mineral from the list of suggested minerals.
